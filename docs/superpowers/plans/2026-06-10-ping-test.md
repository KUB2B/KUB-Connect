# Ping / test server Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** A per-server "Пинг" button in the Settings list that measures TCP connect latency to the server's host:port and shows it inline.

**Architecture:** A new `internal/netping` package does a bounded TCP dial and returns RTT. The app service exposes `Ping(index) PingResultDTO` (dial outside the lock); a bound method surfaces it to the frontend, which renders a ping button + result span per server, caching results by host:port.

**Tech Stack:** Go (`net.DialTimeout`), Wails v2, vanilla TypeScript.

**Spec:** `docs/superpowers/specs/2026-06-10-ping-test-design.md`

// Package netping measures TCP reachability and latency to a host:port.
package netping

import (
	"net"
	"time"
)

// Ping dials addr ("host:port") over TCP and returns the connect round-trip
// time. The dial is bounded by timeout; a successful connection is closed
// immediately.
func Ping(addr string, timeout time.Duration) (time.Duration, error) {
	start := time.Now()
	conn, err := net.DialTimeout("tcp", addr, timeout)
	if err != nil {
		return 0, err
	}
	conn.Close()
	return time.Since(start), nil
}
```

package app

import (
	"errors"
	"net"
	"strconv"
	"time"

	"github.com/zki/vless-client/internal/netping"
)

const pingTimeout = 5 * time.Second

// Ping measures TCP connect latency to the server at the given index. The dial
// runs outside the lock so concurrent pings don't block other operations.
func (s *Service) Ping(index int) PingResultDTO {
	s.mu.Lock()
	if index < 0 || index >= len(s.state.Servers) {
		s.mu.Unlock()
		return PingResultDTO{Error: "неверный сервер"}
	}
	srv := s.state.Servers[index]
	addr := net.JoinHostPort(srv.Host, strconv.Itoa(srv.Port))
	s.mu.Unlock()

	d, err := netping.Ping(addr, pingTimeout)
	if err != nil {
		var ne net.Error
		if errors.As(err, &ne) && ne.Timeout() {
			return PingResultDTO{Error: "таймаут"}
		}
		return PingResultDTO{Error: "недоступен"}
	}
	return PingResultDTO{OK: true, LatencyMs: int(d.Milliseconds())}
}
```
